Welcome to the ML Based AMS Circuit Verification Package

About

The ML Verification Package is a Python package designed to integrate various code components for ML-based verification of AMS circuits. It provides APIs for data processing, machine learning, prediction, evaluation, and feature extraction. The package aims to simplify the development and deployment of ML algorithms such as Multilayer Perceptron (MLP), Random Forest, and Long Short-Term Memory (LSTM) for AMS circuit verification.

Features

  • User-friendly GUI for data input and algorithm selection
  • Data processing API for preparing input data
  • Integration of machine learning algorithms (MLP, Random Forest, LSTM)
  • Prediction API for making predictions on AMS circuits
  • Evaluation API for assessing the performance of ML models
  • Feature extraction API for extracting relevant features from datasets
